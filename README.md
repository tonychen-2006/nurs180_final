# Self-Care Receipt Generator

Frontend-only NURS 180 self-care receipt project.

## How to Run

You do not need VS Code, a code editor, npm, or a terminal.

1. Unzip the project folder.
2. Double-click `index.html`.
3. The app will open in the computer's web browser.

On macOS:

```bash
open index.html
```

With sample data loaded:

```bash
open "index.html?sample=1"
```

No install, server, backend, npm, or internet connection is required.

## Evaluator Handoff

Send the evaluator `self-care-receipt-generator.zip`, or send these files together in one folder:

- `index.html`
- `style.css`
- `script.js`
- `README.md`
- `assets/tony-week1.jpeg`
- `assets/tony-week2.jpeg`
- `assets/tony-week3.jpeg`
- `assets/tony-week4.jpeg`
- `assets/tony-week5.jpeg`
- `assets/tony-week6.jpeg`

The evaluator only needs to unzip the folder and double-click `index.html`. They do not need VS Code.

## PDF Export

1. Click `Load Tony's Data` to fill all six weeks.
2. Click `Download PDF` on a single receipt, or `Download All PDF` for the full set.
3. A `.pdf` file will download directly.

The `Download All PDF` button downloads all logged receipts into one PDF document.
